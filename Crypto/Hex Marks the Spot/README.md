## Challenge Name
Hex Marks the Spot

## Category
Cryptography (Crypto)

## Difficulty
Easy / Beginner

## Description
A string of numbers holds a secret message. By carefully converting the given values, players can reveal the hidden flag.  
This challenge introduces participants to basic hexadecimal encoding commonly used in CTF competitions.

## Challenge Statement
```
43 6C 61 77 43 54 46 7B 48 41 43 4B 45 52 5F 
49 53 5F 41 4C 57 41 59 53 5F 41 5F 48 41 
43 4B 45 52 21 7D
```

## Solution Overview
1. Identify the given values as hexadecimal (base-16).
2. Convert each hex byte into its corresponding ASCII character.
3. Read the decoded text to obtain the flag.

## Final Flag
```
ClawCTF{HACKER_IS_ALWAYS_A_HACKER!}
```

## Learning Outcome
- Understanding hexadecimal (base-16) encoding
- Converting hex values to ASCII text
- Recognizing common encoding patterns in Crypto challenges
